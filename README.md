# EQG-RACE-PLUS
This dataset merge [RACE](https://www.aclweb.org/anthology/D17-1082.pdf) and [EQG-RACE](https://arxiv.org/pdf/2012.06106.pdf) with better format in `jsonl`.

Questions type with General, Cloze, and Specific are split, and the data is sort by article, not sort by question that EQG-RACE format used.
